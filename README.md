🚀 Placement Bridge — Placement Preparation Platform

A full-stack web platform connecting juniors with seniors for structured placement guidance.

🔹 Overview

Placement Bridge is a web-based platform designed to help students prepare for placements through structured, experience-driven guidance.
It connects juniors with seniors who share real interview experiences, preparation strategies, and resources.

🎯 Problem

Students often struggle with:
Where to start placement preparation
Finding authentic resources
Understanding real interview patterns
Guidance is scattered across platforms like YouTube, LinkedIn, and chats, making the process confusing and time-consuming.

💡 Solution

Placement Bridge solves this by providing:
Company-wise interview preparation
Real interview experiences from seniors
Structured learning resources
Mentorship between seniors and juniors

👥 Users
Juniors: Access resources, questions, mentorship
Seniors: Share experiences and guidance
Admin: Manage platform and content

🏗️ Architecture Overview
Frontend (React) → Backend API (Node.js + Express) → Database (PostgreSQL)

⚙️ Tech Stack
Frontend: React, Vite, TypeScript, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL (Neon)
Authentication: Express Session, Passport.js
Realtime: WebSocket
UI Libraries: Shadcn UI, Radix UI, Framer Motion

✨ Features
Secure authentication (login/signup)
Company-wise interview preparation
Senior experience sharing (questions, tips)
Mentorship connection system
Resource sharing (notes, links, guides)
Discussion forum for queries
User profiles with skills and placement status

🔁 Workflow
User → Signup/Login → Select Role (Senior/Junior) → Dashboard → Access Features
